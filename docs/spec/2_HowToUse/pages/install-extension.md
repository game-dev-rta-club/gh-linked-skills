# Linked SkillsのInstall

Linked Skills本体をGitHub CLI extensionとしてinstallする。

前提: macOSまたはLinux、`gh`、Git。

```bash
gh extension install game-dev-rta-club/gh-linked-skills
```

WindowsとGHESは正式対応しない。
