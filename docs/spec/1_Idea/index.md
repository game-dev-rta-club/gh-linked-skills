# Idea

Linked Skillsは、projectで育ったskillをsourceへ戻し、別のprojectでも使えるようにする。

Skillを配るだけではなく、使用中に得た改善を次の利用へつなげるためのtoolである。

- `skill`: agentが使う手順や補助fileをまとめたfolder
- `local skill`: project内で実際に使い、編集するskill
- `source`: skillを共有するGitHub repository内の配布元
